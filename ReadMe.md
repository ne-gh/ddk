# NeKernel DDK (Mark 1)

Use this to write drivers for NeKernel.

## Installation:

- MinGW/GCC
- Git
- OpenBTB

### Start by cloning the DDK:

```
git clone git@github.com:ne-gh/driver-kit.git
cd ddk
```

### And then select the makefile to execute:

```
cd dev/LibDDK
btb ddk.json
```
